## 🏆 Key Projects

### 🌍 **Global Economic Dashboard** *(2025)*
Interactive platform monitoring GDP, inflation, unemployment & interest rates across 50+ countries with automated forecasting.
- **Tech:** Python, Streamlit, Plotly, Facebook Prophet, GitHub Actions
- **Live:** [economic-dashboard-bahf3zb69es2aliamubamr.streamlit.app](https://economic-dashboard-bahf3zb69es2aliamubamr.streamlit.app) | [GitHub](https://github.com/kashiruddinshaik/economic-dashboard)

### 🚦 **Adaptive Traffic Signal Control** *(2023)*
Reinforcement learning system using Q-Learning to dynamically optimize traffic signal timers with measurable CO₂ reduction.
- **Tech:** Python, Q-Learning, Reinforcement Learning, HERE Maps API
- **GitHub:** [TrafficControlAlgo](https://github.com/Kashiruddinshaik/TrafficControlAlgo)

### 📊 **Customer Churn Prediction System** *(2024)*
ML web app trained on real telecom data using Logistic Regression, Random Forest & XGBoost with GridSearchCV tuning.
- **Tech:** XGBoost, Scikit-learn, Streamlit, Python
- **Live:** [churn-prediction-6s4f.onrender.com](https://churn-prediction-6s4f.onrender.com/) | [GitHub](https://github.com/kashiruddinshaik/churn-prediction-system)

### 🍕 **Restaurant Full-Stack Platform** *(2025)*
Production website for Wellesley NorthEnd Pizza — digital menus, ordering flow, dynamic campaign management. Live with real customers.
- **Tech:** Django, JavaScript, HTML/CSS, Python
- **Live:** [wellesleynorthendpizza.com](https://wellesleynorthendpizza.com)
